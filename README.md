# DSA MASTER

> AI-Powered DSA Learning & Placement Preparation Platform

DSA MASTER is a comprehensive Data Structures & Algorithms learning platform designed to take students from beginner level to placement-ready level through structured roadmaps, topic-wise learning, company-specific preparation, curated question banks, analytics, revision tools, and AI-assisted tutoring.

⚠️ **Transparency Notice:**  
This project was developed primarily using AI coding agents under human supervision. The architecture, feature planning, debugging, testing, and iterative improvements were guided, reviewed, and refined by the project owner.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/DSA-MASTER.git

cd DSA-MASTER
```

### Frontend

```bash
npm install

npm run dev
```

### Backend

```bash
mvn spring-boot:run
```

or

```bash
npm install

npm run start
```
